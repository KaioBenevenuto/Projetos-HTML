Áudio e vídeo

<Video>

  • src → É usado para colocar o link do vídeo

  • controls → Serve para fazer com que os botões de controle como pause, volume dentre outros apareçam na tela do vídeo

  • Dependendo do navegador pode não funcionar. Nesse caso pode por uma mensagem dizendo que o navegador não suporta o vídeo e deixar um link para que o usuário possa baixar o vídeo

  • <source> → Funciona como um container, pode ser usado para coloocar mais de um vídeo
    • src
    • type → É usado no source para especificar qual é o tipo de vídeo

  • Uma outra forma de adicionar vídeo no HTML é usando serviços de terceiros, como youtube por exemplo

Propriedades

width | height
Modificam a largura e altura do vídeo

autoplay
Inicia a página tocando o vídeo

preload="auto"
Antes de dar play no vídeo ele já vai carregando

preload="metadata"
Carrega só o básico do vídeo, como o tamanho dele por exemplo

preload="none"
Não carrega nada antes de dar play

loop
Faz com que quando o vídeo chegue no final, ele comece de novo

muted
O vídeo começa mutado

poster="link da imagem"
Coloca uma imagem de fundo na tela do vídeo, antes dele começar

Imagem rasterizada X Imagem vetorizada

Benefícios da imagem vetorizada

  • Mais leve
  • Mais detalhada
  • Maior acessibildade e SEO
  • Pode ser editada via CSS ou atributos

Desvantagens

  • Pode ser mais comlicado de trabalhar
  • Quanto mais complexa a imagem, mais trabalho para o navegador
  • Navegadores mais antigos não possuem suporte a essa tag

Para fotografias, é recomendado o uso de imagens rasterizadas
